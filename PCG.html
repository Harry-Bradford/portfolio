PCG

Using Unreal Engine's Procedural Content Generation, I have created tools to build environments in a matter of minutes.

Inside of a volume, grass, rocks, logs, and trees can be generated dynamically. The density of each type of mesh can be varied, and the overall size of the volume can be easily adusted.

We can also procedurally generate meshes on top of other meshes, in this example, we can change the amount of mushrooms growing on a log.

Splines can also be used to set the boundaries of the content generated. Different types of content can be made to intersect, like this path cutting through the trees.
